# Sistema-de-Gestion-de-Productos
Aplicación de escritorio desarrollada en C# (Windows Forms) con SQL Server, orientada a la gestión de productos de indumentaria (remeras, calzas, buzos, pantalones, camperas, etc.).
El sistema permite visualizar y controlar el stock disponible (cantidad, talles, colores, marcas, entre otros atributos), agregar nuevos productos con inserción directa en la base de datos y actualizar el listado en tiempo real.
También brinda la posibilidad de modificar la información de cualquier producto y realizar una baja lógica de los mismos desde la base de datos.
